# Netlify 배포 가이드

소노 아임레디 스마트케어 랜딩 페이지를 **Netlify**를 통해 무료로 쉽게 배포하는 방법입니다.

## 방법 1: 드래그 앤 드롭 (가장 빠름)

복잡한 설정 없이 파일을 끌어다 놓기만 하면 즉시 웹사이트가 만들어집니다.

1.  **Netlify 접속**: [https://app.netlify.com/](https://app.netlify.com/) 에 접속하여 로그인합니다. (계정이 없다면 'Sign up'으로 가입)
    *   GitHub, GitLab, Bitbucket 아이디나 이메일로 가입 가능합니다.
2.  **Sites 메뉴 이동**: 로그인 후 좌측 메뉴에서 **'Sites'**를 클릭합니다.
3.  **폴더 업로드**:
    *   화면 하단에 보면 **"Drag and drop your site output folder here"**라는 점선 박스 영역이 있습니다.
    *   작업하신 **프로젝트 폴더 전체**(`2. sono_smart2`)를 마우스로 끌어서 저 박스 안에 놓습니다.
    *   *주의: `index.html` 파일 하나만 넣는 게 아니라, 이미지가 포함된 '폴더 전체'를 넣어야 합니다.*
4.  **배포 완료 확인**:
    *   업로드가 끝나면 무작위 이름(예: `happy-otter-123456.netlify.app`)으로 사이트 주소가 생성됩니다.
    *   주소를 클릭하여 사이트가 정상적으로 보이는지 확인합니다.

---

## 방법 2: GitHub 연동 (추천)

코드를 수정할 때마다 자동으로 배포되게 하려면 이 방법을 씁니다. (GitHub 사용 경험이 있을 경우)

1.  현재 프로젝트를 GitHub 저장소(Repository)에 올립니다.
2.  Netlify에서 **'Add new site'** -> **'Import from existing project'**를 클릭합니다.
3.  **GitHub**를 선택하고, 방금 올린 저장소를 선택합니다.
4.  **'Deploy Site'** 버튼을 누르면 끝입니다.

---

## ⚠️ 배포 전 체크리스트

1.  **메인 파일 이름**: 반드시 `index.html`이어야 합니다. (현재 맞습니다)
2.  **이미지 경로**:
    *   이미지 파일명에 한글이 있거나 공백이 있으면 가끔 오류가 날 수 있습니다. (현재 `partner_logo_clean.png` 등 영문으로 잘 되어 있어 문제 없을 것입니다.)
    *   코드 상에서 이미지 경로가 절대 경로(`C:/Users/...`)가 아닌 **상대 경로**(`logo.png`, `images/logo.png`)로 되어 있어야 합니다. (현재 잘 되어 있습니다.)
3.  **구글 시트 연동**:
    *   Netlify는 `https`(보안 서버)를 사용하므로, 구글 시트 연동 기능이 로컬(`file://`)보다 훨씬 안정적으로 잘 작동할 것입니다.

이제 방법 1을 시도해 보시고, 주소가 나오면 알려주세요! 주소도 예쁘게 변경 가능합니다.
